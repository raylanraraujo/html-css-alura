Computadores também tem um terreno, que nesse caso é chamado de MEMÓRIA. Nela é possível colocar “carros” que nesse caso são os DADOS. Se colocar os dados de maneira desenfreada é provável que esses dados acabem se perdendo. Portanto, é preciso que dentro da memória do computador (o terreno) sejam definidos espaços delimitados (as vagas) para receber os dados (os carros). Chamamos esses espaços delimitados de variáveis e para que possa ser “usada” é preciso que ela contenha um identificador (um nome). Os dados atribuídos a cada variável identificada é chamado valor. Quando um valor é colocado dentro de uma variável dizemos que foi feito uma atribuição. Variáveis servem para guardar dados.

  • vaga a1 = carro1 // um sinal de igual no JS deve-se ler como ”recebe”
  • vaga a1 = carro2 // “Vaga a1 recebe carro2” → para que isso possa acontecer é preciso que tire o carro1 da vaga
  • vaga a1 = null // “Vaga a1 recebe nada dentro → com isso eu tiro o que está dentro da vaga

Podemos utilizar tanto a palavra var quanto a palavra let para criar variáveis
